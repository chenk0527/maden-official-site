# 马登工装官网开发 Brief

## 品牌信息
- **品牌名**：马登工装 MADEN
- **英文**：MADEN WORKWEAR
- **创立时间**：2008年，3人创立；2011年马登工装品牌正式诞生
- **定位**：美式复古工装 (Americana Workwear)
- **Slogan 建议**："BORN TO WORK. MADE TO LAST." 或 "美式复古工装，始于2008"
- **总部**：重庆
- **实体店**：重庆5家 + 成都2家
- **品牌矩阵**：男装(马登工装) 65.8% | 鞋(马登男鞋) 22.5% | 女装(马登女装) 11.6%
- **全渠道70+店铺**：抖音/淘宝/天猫/京东/拼多多/得物/实体店/海外TikTok

## 设计要求
1. **现代化、炫酷**：参考 Nike、Carhartt WIP、Stüssy 等潮流品牌官网风格
2. **响应式**：手机端和PC端都要完美适配
3. **完成度要高**：不能是半成品，要像真正的品牌官网
4. **纯前端**：HTML + CSS + JavaScript，不需要后端
5. **SEO友好**：Schema标记、语义化HTML、Open Graph标签
6. **动效**：滚动动画、视差效果、平滑过渡，要有质感
7. **中文为主**：面向国内用户，品牌名英文

## 页面结构

## 注意事项
- 图片URL可能有签名过期问题，如果不能加载，用高质量的占位图（unsplash上的工装/时尚图片代替）
- 如果图片有跨域问题，使用 CSS background-image 或带 referrerpolicy="no-referrer" 
- **一定要用 referrerpolicy="no-referrer" 加载抖音图片，否则会403**
- Logo 如果加载失败，用 CSS 文字 logo 代替（"MADEN" 粗体字）
- 颜色方案：深色/军事色调为主（#1a1a1a, #2d2d2d, #c49a6c/驼色, #4a5a3a/军绿）
- 字体：思源黑体 + 英文用 Oswald 或 Bebas Neue（硬核工装感）

## 技术栈
- 纯 HTML5 + CSS3 + Vanilla JavaScript
- CSS Grid + Flexbox 布局
- CSS 自定义属性（变量）
- Intersection Observer API（滚动动画）
- 不需要框架，保持轻量和SEO友好
- 加载 Google Fonts CDN（使用国内镜像 fonts.loli.net）
