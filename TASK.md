# Task: Build MADEN Official Website

Read BRIEF.md for full details. Build a complete, production-ready website.

## Files to Create
1. `index.html` - Single page with all sections
2. `styles.css` - All styles
3. `main.js` - Animations and interactions

## Design
- Dark theme (#1a1a1a bg, #c49a6c gold accent, #4a5a3a military green)
- Oswald + Noto Sans SC fonts
- Scroll animations (Intersection Observer)
- Parallax hero section
- Responsive (mobile first)

## Sections (in order)
1. Navigation (sticky, transparent → solid on scroll)
2. Hero (full viewport, brand name MADEN, slogan, CTA)
3. Brand Story (2008-2026 timeline, 16 years)
4. Product Categories (4 cards: Jackets, Pants, Shoes, Womenswear)
5. Lookbook Gallery (product images grid)
6. Brand Stats (16 years / 70+ stores / 7 physical stores)
7. Shopping Channels (icons linking to Tmall, Douyin, JD, PDD, Dewu)
8. Physical Stores (Chongqing 5 + Chengdu 2)
9. Footer
